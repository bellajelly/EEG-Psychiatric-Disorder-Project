# EEG-Psychiatric-Disorder-Project
For my capstone project I'm exploring whether EEG frequency band features can classify psychiatric disorders, and more importantly, whether the patterns a machine learning model learns align with what neuroscience already predicts about brain oscillations in conditions like depression, anxiety, and schizophrenia.
I chose this topic because it sits at the intersection of two things I care about deeply: computational methods and the neuroscience of the brain. Decades of research have identified EEG signatures associated with specific disorders, suppressed alpha in depression, elevated theta in anxiety, disrupted gamma in schizophrenia, but those findings rarely make it into practical tools. I want to see if a classifier can operationalize that knowledge, and whether the model's learned signal actually agrees with the literature.
The dataset is the EEG Psychiatric Disorders Dataset from Kaggle, which contains pre-extracted frequency band power features across delta, theta, alpha, beta, and gamma bands for patients diagnosed with mood disorders, anxiety disorders, schizophrenia, addictive disorders, and others. The features are already computed, which means I can focus on the modeling and interpretation rather than raw signal processing.
My primary technique will be a Random Forest classifier, which I chose specifically because its feature importance scores let me ask which frequency bands matter most for classification. I'll also run an SVM for comparison. The most interesting part of the project won't be the accuracy number, it'll be whether the feature importances match what the neuroscience literature predicts, and what to make of it when they don't.
